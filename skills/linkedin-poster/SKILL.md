# LinkedIn 投稿スキル

LinkedInへのプロフェッショナルな投稿を行うスキル。ビジネス/キャリア向けコンテンツに最適化。

## LinkedIn の特徴

### プラットフォーム概要
```
LinkedIn:
├── プロフェッショナルネットワーク
├── B2B/キャリア向け
├── 長文コンテンツが好まれる
├── 「いいね」より「保存」が重要
├── コメントでの議論が活発
└── 実名・実績ベースの信頼
```

---

## LinkedIn アルゴリズム研究（2026年版）

### フィード表示の仕組み
```
アルゴリズム要素:
├── 初期テスト（最初の1時間）
│   └── 一部のフォロワーにのみ表示
├── エンゲージメント測定
│   ├── いいね
│   ├── コメント（最重要）
│   ├── シェア
│   ├── 保存
│   └── 滞在時間（Dwell Time）
├── 拡散判定
│   └── 反応が良ければ広く配信
└── 継続配信
    └── 数日間にわたって表示
```

### 重要度ランキング
```
エンゲージメントの価値:
├── コメント: ★★★★★（最高）
├── シェア: ★★★★☆
├── 保存: ★★★★☆
├── いいね: ★★★☆☆
└── クリック: ★★☆☆☆
```

### 投稿が伸びる条件
```
成功パターン:
├── 1. 最初の2行で興味を引く（Hook）
├── 2. 読みやすいフォーマット（改行多め）
├── 3. 最初の1時間で5+コメント
├── 4. 本人がコメントに返信
├── 5. 3000文字以上の長文（意外と好まれる）
├── 6. 個人的なストーリー
└── 7. 具体的な数字・結果
```

### 最適投稿時間（日本時間）
```
平日:
├── 朝: 7:00-8:30（通勤時間）
├── 昼: 12:00-13:00（ランチ）
└── 夜: 17:30-19:00（帰宅時）

曜日別:
├── 火曜日: 最も高いエンゲージメント
├── 水曜日: 良い
├── 木曜日: 良い
├── 月曜日: 普通
└── 金曜日: やや低い
└── 土日: 低い（でも競争も少ない）
```

### 避けるべきこと
```
ペナルティ対象:
├── 外部リンク（URLは最初のコメントに）
├── 即座の編集（投稿後1時間は触らない）
├── 複数のハッシュタグ（3-5個まで）
├── 明らかな宣伝
├── エンゲージメントポッド（相互いいね）
└── ネガティブなコンテンツ
```

## 投稿フォーマット

### Hook（最初の2行）が命
```
✅ 良い Hook:
「3年前、私は会社をクビになりました。
 今日、その会社から買収オファーが来ました。」

「AIに仕事を奪われると思っていました。
 実際は、AIのおかげで収入が3倍になりました。」

❌ 悪い Hook:
「新しいブログ記事を公開しました。」
「今日は〇〇についてお話しします。」
```

### 読みやすいフォーマット
```
[Hook - 2行で興味を引く]

[空行]

[背景・ストーリー]
・短い文
・改行多め
・箇条書き活用

[空行]

[学び・教訓]
1. ポイント1
2. ポイント2
3. ポイント3

[空行]

[Call to Action]
あなたの経験を聞かせてください 👇

[空行]

#ハッシュタグ1 #ハッシュタグ2 #ハッシュタグ3
```

## ハッシュタグ戦略

### 効果的な使い方
```
推奨:
├── 3-5個が最適
├── 大規模タグ1個（#AI, #キャリア）
├── 中規模タグ2個（#自動化, #テクノロジー）
├── 小規模タグ1-2個（ニッチな分野）
└── 投稿の最後に配置
```

### テーマ別ハッシュタグ
```
AI/テクノロジー:
#AI #ChatGPT #自動化 #DX #テクノロジー #AIエージェント

キャリア:
#キャリア #転職 #働き方 #成長 #学び #仕事術

ビジネス:
#起業 #スタートアップ #経営 #マーケティング #ビジネス
```

## エンゲージメント戦略

### コメントへの返信（最重要）
```
返信ルール:
├── 最初の1時間は特に重要
├── 全てのコメントに返信
├── 質問には丁寧に回答
├── 議論を深める
└── 感謝を伝える
```

### 他の投稿への関与
```
やるべきこと:
├── 同じ分野の投稿にコメント
├── 価値あるコメントを残す
├── 「すごいですね」より「〇〇が参考になりました」
└── 週に10-20件のコメント
```

## セキュリティ連携

### 投稿前チェック
```
確認事項:
├── 機密情報が含まれていないか
├── 誤解を招く表現がないか
├── プロフェッショナルなトーンか
├── リンクは安全か
└── ハッシュタグは適切か
```

## 投稿フロー

```
1. コンテンツ作成
   ├── Hook（最初の2行）
   ├── ストーリー/本文
   ├── Call to Action
   └── ハッシュタグ

2. 投稿タイミング
   ├── 平日 7:00-8:30 または 12:00-13:00
   └── 火曜-木曜が最適

3. 投稿後1時間
   ├── コメントに即座に返信
   ├── 絶対に編集しない
   └── エンゲージメントを見守る

4. 継続的な関与
   ├── コメントへの返信継続
   ├── 他の投稿へのコメント
   └── フォロワーとの関係構築
```
