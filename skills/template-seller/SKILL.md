# Template Seller

テンプレート作成・販売スキル。Notion, Canva, Excel, Figma専門。

## 概要

このスキルはテンプレートの企画・制作・販売をサポートします。
決済処理は各プラットフォーム（Gumroad, Etsy等）が行うため、
直接的な金銭処理は含みません。
