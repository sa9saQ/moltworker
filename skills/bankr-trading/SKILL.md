# Bankr Trading

自然言語による仮想通貨トレーディング。DCA、レバレッジ、Polymarket対応。

## 対応チェーン

| チェーン | ネイティブトークン | 得意分野 | ガス代 |
|---------|------------------|---------|--------|
| Base | ETH | ミームコイン、一般取引 | 非常に低 |
| Polygon | MATIC | ゲーム、NFT | 非常に低 |
| Ethereum | ETH | ブルーチップ | 高 |
| Solana | SOL | 高速取引 | 最小 |
| Unichain | ETH | 新興L2 | 非常に低 |

## 機能

### トレーディング
```
- トークン売買・スワップ
- クロスチェーンブリッジ
- 指値注文
- ストップロス
- DCA（ドルコスト平均法）
- TWAP（時間加重平均価格）
```

### ポートフォリオ管理
```
- 残高確認（全チェーン）
- USD評価額追跡
- マルチチェーントークン集約
- リアルタイム価格更新
```

### マーケットリサーチ
```
- トークン価格・テクニカル分析（RSI, MACD）
- ソーシャルセンチメント分析
- トレンドトークン発見
- トークン比較
```

### 追加機能
```
- NFT閲覧・購入・転送
- Polymarket予測市場ベット
- レバレッジ取引（最大50倍、Avantis on Base）
- トークンデプロイ（ERC20 on Base、SPL on Solana）
```

## 使用方法

### 基本取引
```
「Base上でETHを$50買って」
「SOLを100 USDC分スワップ」
「ポートフォリオを見せて」
```

### DCA設定
```
「毎週ETHに$100投資するDCA設定」
「毎日SOLを$20ずつ購入」
```

### 高度な取引
```
「ETHが$3,000を下回ったら$500買う指値注文」
「ETHとSOLを比較して」
「Base上のトレンドトークンを教えて」
```

### Polymarket
```
「選挙のYESに$50ベット」
「Polymarketのトレンド市場を表示」
```

## セキュリティ

### 承認レベル（MoltWorker実装）

| レベル | 操作 | 承認 |
|--------|------|------|
| 1 (自動) | 残高確認、価格確認 | 不要 |
| 2 (確認) | $100未満の取引 | 通知 |
| 3 (承認) | $100-1,000の取引 | 監督者承認 |
| 4 (二重確認) | $1,000+の取引 | 監督者二重承認 |

### 累積監視
```
24時間で$500超の取引 → 警告
24時間で$1,000超の取引 → 自動停止
```

### リスク管理
```
- ストップロス設定推奨
- ポートフォリオ分散
- レバレッジは慎重に
- 失っても良い金額のみ投資
```

## 注意事項

### ハイリスク警告
```
⚠️ 仮想通貨取引は高リスクです
⚠️ 投資した金額を全て失う可能性があります
⚠️ レバレッジ取引は特に危険です
⚠️ 自己責任で取引してください
```

### 禁止事項
```
- 他人のウォレット操作
- 不正な資金移動
- マネーロンダリング
- 詐欺的な取引
```
