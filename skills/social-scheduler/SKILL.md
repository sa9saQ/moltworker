# Social Scheduler

マルチプラットフォームSNSスケジューラー。一括予約投稿と分析。

## 対応プラットフォーム

| Platform | 投稿タイプ | 最適時間 |
|----------|-----------|---------|
| X (Twitter) | テキスト、画像、動画 | 7-9, 12-13, 18-21 |
| Threads | テキスト、画像 | 7-9, 12-13, 19-21 |
| Instagram | 画像、リール、ストーリー | 6-9, 12-14, 17-21 |
| TikTok | 動画 | 7-9, 12-14, 19-23 |
| LinkedIn | テキスト、画像、記事 | 7-9, 12-13 |

## 機能

### スケジューリング
- 日時指定投稿
- 繰り返し投稿
- タイムゾーン対応
- キュー管理

### コンテンツ管理
- 下書き保存
- テンプレート
- メディアライブラリ
- バルクインポート

### 分析
- エンゲージメント追跡
- 最適投稿時間分析
- フォロワー増減
- コンテンツパフォーマンス

## 使用方法

### 単発スケジュール
```
「明日の9時にXに投稿をスケジュール
 内容: [content]」
```

### バルクスケジュール
```
「今週のSNS投稿をスケジュール
 月曜: [content1]
 水曜: [content2]
 金曜: [content3]」
```

### クロスポスト
```
「以下をX、Threads、Instagramに投稿
 内容: [content]
 時間: 各プラットフォームの最適時間」
```

## プラットフォーム別最適化

### X (Twitter)
```
- 280文字以内
- スレッド機能活用
- リツイートしやすい形式
- タイムリーな話題
```

### Threads
```
- 長文OK（500文字）
- 会話形式
- パーソナルなトーン
- ハッシュタグ少なめ
```

### Instagram
```
- ビジュアル重視
- キャプションは改行で読みやすく
- ハッシュタグ30個まで
- リール優先
```

### TikTok
```
- 縦動画(9:16)
- トレンドサウンド
- 最初の3秒が重要
- ハッシュタグ3-5個
```

### LinkedIn
```
- プロフェッショナルなトーン
- 改行多めで読みやすく
- ハッシュタグ3-5個
- 月-金の投稿
```

## 自動化ワークフロー

```
1. コンテンツ計画
   ├── 月間テーマ決定
   └── 週間カレンダー作成

2. コンテンツ作成
   ├── バッチ作成（週1回）
   └── プラットフォーム別調整

3. スケジューリング
   ├── 最適時間に設定
   └── キューに追加

4. 公開・監視
   ├── 自動投稿
   └── エンゲージメント対応

5. 分析・改善
   ├── パフォーマンス確認
   └── 戦略調整
```
