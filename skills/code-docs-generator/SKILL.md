## 出力例

### OpenAPI仕様
```yaml
openapi: 3.0.0
info:
  title: API Name
  version: 1.0.0

paths:
  /users:
    get:
      summary: Get all users
      responses:
        '200':
          description: Success
          content:
            application/json:
              schema:
                type: array
                items:
                  $ref: '#/components/schemas/User'
```

### TypeDoc出力

```markdown
### Returns

`Promise<User>` - The user object

### Example

\`\`\`typescript
const user = await getUserById('123');
\`\`\`
```
