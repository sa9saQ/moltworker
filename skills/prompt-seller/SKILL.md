## プロンプト作成ガイド

### 品質基準

```
必須要素:
├── 明確な目的
├── 再現性（同じ結果が出る）
├── カスタマイズ性
├── 詳細な説明書
└── 使用例
```

### テキストプロンプト構造

```
[Role/Persona]
あなたは〇〇の専門家です。

[Context]
背景情報と制約条件

[Task]
具体的なタスク

[Format]
出力形式の指定

[Examples]
入出力の例

[Variables]
{{カスタマイズ可能な変数}}
```

### 画像プロンプト構造

```
[Subject] 主題
[Style] スタイル（写真、イラスト、3D等）
[Mood] 雰囲気（明るい、暗い等）
[Lighting] 照明
[Camera] カメラ設定
[Details] 詳細指定
[Negative] 除外要素
```
