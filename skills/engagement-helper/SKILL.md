## エンゲージメント戦略

### 日々の交流
```yaml
毎日:
  - 自分の投稿へのリプライに全返信
  - 関連アカウントの投稿にいいね/リプライ
  - 新規フォロワーへの挨拶

週1-2回:
  - 質問投稿でフォロワーと対話
  - 他ユーザーの投稿を引用RT
```

### 関係性レベル
```
Level 1: 一度交流した
Level 2: 複数回交流
Level 3: 定期的に交流
Level 4: 相互フォロー＋頻繁交流
```

## 困った時の対応

### ネガティブコメント
```yaml
無視すべき:
  - 明らかな荒らし
  - 攻撃的な言葉
  - 個人攻撃

対応すべき:
  - 建設的な批判 → 感謝して受け止める
  - 誤解 → 丁寧に説明
  - 質問形式の批判 → 真摯に回答
```

### 対応テンプレート
```
建設的批判:
「ご指摘ありがとうございます。確かにその点は改善の余地がありますね。参考にさせていただきます。」

誤解:
「ご質問ありがとうございます。説明が不十分でしたね。実は[説明]ということでした。」

攻撃的（返信する場合）:
「ご意見ありがとうございます。」（それ以上反応しない）
```
