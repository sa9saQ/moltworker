## 自動挿入ルール

### Note記事

```yaml
挿入位置:
  - 記事冒頭（関連商品紹介時）
  - 記事中盤（解説と共に）
  - 記事末尾「関連商品」セクション

フォーマット:
  "▼ 関連商品
   [商品名]（PR）
   [リンク]"
```

### X投稿

```yaml
ルール:
  - 1投稿につき1リンクまで
  - 本文後に配置
  - PR表記必須

フォーマット:
  "[本文]

   👉 [商品名]（PR）
   [短縮URL]"
```

### Threads投稿

```yaml
ルール:
  - リンクは投稿末尾
  - 自然な文脈で紹介
  - PR表記必須
```

## PR表記ルール（MUST）

### 日本の法規制対応

```
必須表記:
├── 「PR」
├── 「広告」
├── 「アフィリエイト」
├── 「※リンクから購入すると報酬が発生します」
└── ステマ規制法対応

配置:
├── リンクの直前または直後
├── 投稿の目立つ位置
└── 記事の場合は冒頭にも記載
```

### NGパターン

```
❌ PR表記なしでリンク設置
❌ 「おすすめ！」だけでPR隠す
❌ 記事最後に小さく「PRあり」
```
