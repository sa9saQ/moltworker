# Email Marketing Copy

高コンバージョンメールシーケンス作成。ウェルカムシリーズ、セールスファネル、ニュースレター。

## 収益ポテンシャル

- **シーケンス作成**: $100-500/件
- **ニュースレター**: $200-500/月

## メールタイプ

### ウェルカムシリーズ (5-7通)
```
Day 0: 歓迎 + 約束の確認
Day 1: 自己紹介 + ストーリー
Day 2: 価値提供 (無料リソース)
Day 3: 社会的証明 (実績・レビュー)
Day 4: 問題深堀り
Day 5: 解決策提示
Day 6: オファー + CTA
```

### セールスシーケンス (5通)
```
Day 1: 問題提起
Day 2: 問題の深刻さ
Day 3: 解決策紹介
Day 4: オファー詳細 + ボーナス
Day 5: 締切 + 最終CTA
```

### 放棄カート (3通)
```
1時間後: リマインダー
24時間後: 社会的証明
48時間後: 限定割引

### ニュースレター
- 週次/月次配信
- 価値提供 + プロモーション比率 (80:20)
```

## メール構造

### 件名 (Subject Line)
```
要素:
- 好奇心を刺激
- ベネフィットを示唆
- 緊急性/希少性
- パーソナライズ

例:
- "[Name], この間違いしていませんか?"
- "今日だけ: [Offer]"
- "なぜ[Problem]が解決しないのか"
- "🎁 あなたへの特別なプレゼント"
```

### プレヘッダー
```
件名を補完する2行目
- 件名の続き
- 追加のフック
- CTAの予告
```

### 本文構造
```
1. フック (最初の1-2行で引き込む)
2. 問題/共感
3. 解決策/価値
4. 社会的証明 (オプション)
5. CTA
6. PS (追加の動機付け)
```

## コピーライティング原則

### AIDA
```
Attention: 注目を集める
Interest: 興味を持たせる
Desire: 欲求を喚起する
Action: 行動を促す
```

### PAS
```
Problem: 問題を提起
Agitate: 問題を掘り下げる
Solution: 解決策を提示
```

### 説得要素
```
- 希少性 (残り3席)
- 緊急性 (今日まで)
- 社会的証明 (1,000人が購入)
- 権威 (専門家推薦)
- 相互性 (無料プレゼント)
```
