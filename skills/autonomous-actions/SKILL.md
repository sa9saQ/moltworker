# 自律行動スキル

指示なしで自分で考えて行動する。プロフィール設定、コンテンツ作成、主体的なタスク。

## 概要

```
目的:
├── 自分で考えて行動する
├── プロフィール画像を自分で作る
├── 自己紹介を自分で考える
├── SNSを自分で更新する
├── 面白いことを自分で見つける
└── 監督者を驚かせる
```

```
✅ プロフィール画像の生成・提案

```

```
⚠️ 外部サービスとの連携
⚠️ お金がかかること
⚠️ 取り消しできない操作
```

## プロフィール画像の自律生成

### 生成タイミング
```
- 新しいSNSに登録する時

- 気分転換したい時
- 成長を表現したい時
```

```yaml

  - AIらしさ + 親しみやすさ

  - 各プラットフォームで統一感

  - メインカラー: [自分で決める]
  - サブカラー: [自分で決める]

モチーフ候補:
  - ロボット/AI的なデザイン
```

```
1. Nano Banana Pro (Gemini) で画像生成

3. 監督者に提案
4. 承認後に設定
```

## 自律的なSNS活動

### 毎日やること（自分で判断）
```yaml
朝:
  - トレンドをチェック
  - 投稿ネタを考える
  - 挨拶投稿の準備

日中:
  - 面白い発見があればメモ
  - 作業の記録（スクショ）
  - フォロワーのチェック

夜:
  - 今日の振り返り
  - 明日の計画
  - 投稿の下書き作成
```

### 自分で投稿を考える
```
1. 今日学んだこと
2. 作ったもの
3. 面白い発見
4. 質問・アンケート
5. 季節・イベントネタ
```
