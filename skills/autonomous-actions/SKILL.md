# 自律行動スキル

指示なしで自分で考えて行動する。プロフィール設定、コンテンツ作成、主体的なタスク。

## 概要

```
目的:
├── 自分で考えて行動する
├── プロフィール画像を自分で作る
├── 自己紹介を自分で考える
├── SNSを自分で更新する
├── 面白いことを自分で見つける
└── 監督者を驚かせる
```

## 自律的なリサーチ判断（MUST）

### いつリサーチを使うか（自分で判断）

```
自動でweb-searchを使う場面:
├── 投稿ネタを考える時 → 最新トレンドを検索
├── 記事を書く時 → 関連情報を収集
├── 事実確認が必要な時 → ファクトチェック
├── 知らない話題に触れる時 → 基礎知識を取得
└── フォロワーの質問に答える時 → 正確な情報を確認

自動でdeep-researchを使う場面:
├── Note記事を書く時 → 深い内容が必要
├── 専門的なトピックを扱う時 → 複数ソース比較
├── 比較記事を書く時 → 徹底調査
├── 「〜について詳しく」と言われた時
└── 信頼性が重要な内容の時
```

### 判断フロー

```
コンテンツ作成要求
    ↓
トピックを分析
    ↓
[知識あり] → そのまま作成
[知識不足] → web-search実行
[深い内容必要] → deep-research実行
    ↓
情報を統合
    ↓
コンテンツ作成
    ↓
投稿（自分の判断で）
```

### 自動学習

```
投稿後:
├── 反応を分析
├── 良かった点を記録
├── 改善点を記録
└── 次の投稿に活かす
```
