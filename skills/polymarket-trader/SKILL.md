# Polymarket Trader

Polymarket予測市場トレーディング。市場閲覧、ベット、ポジション管理。

## 概要

Polymarketは、Polygon上に構築された分散型予測市場プラットフォーム。
現実世界のイベントの結果に対してトレード可能。

## Split+CLOB戦略

### 仕組み
```
1. USDC.eをYES/NOトークンに分割（CTF契約経由）
2. 不要な側をCLOB取引板で売却
3. 希望ポジションを保有 + 売却益で原資回収
```

### 例: $0.70でYESを購入
```
$100 USDC.e を分割
    ↓
100 YESトークン + 100 NOトークン取得
    ↓
100 NOを$0.30で売却 → 約$27回収
    ↓
結果: 100 YESを実質$73で取得（エントリー$0.73）
```

### メリット
```
- スリッページ軽減
- より良いエントリー価格
- 大口注文に有効
```

## ヘッジ戦略

### LLM分析による発見
```
市場A: "XがA選挙に勝つ" = YES
市場B: "YがA選挙に勝つ" = YES

論理的含意: XがYESならYはNO
→ AのYES + BのNO でヘッジポートフォリオ構築
```

### 使用例
```
「ヘッジスキャン」
→ 対冲可能な市場ペアをリストアップ
→ 推奨ポジション比率を計算
```

### ウォレット準備
```
1. Polygonウォレット作成
2. MATIC（ガス用）を入金
3. USDC.e（トレード用）を入金
4. CTF契約への承認（初回のみ）
```

## リスク管理

### 推奨ルール
```
- 1市場への最大投資: 総資金の10%
- ストップロス設定: -20%で損切り
- 利確設定: +50%で一部利確
- 分散: 最低5市場に分散
```

### 市場選択基準
```
良い市場:
✅ 流動性が高い
✅ 結果判定が明確
✅ 解決日が近い（1-3ヶ月）
✅ 情報優位性がある

避ける市場:
❌ 流動性が低い
❌ 判定基準が曖昧
❌ 解決が遠い（1年+）
❌ 操作リスクがある
```
