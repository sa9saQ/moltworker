# Course Creator

オンラインコース作成・販売スキル。カリキュラム設計、動画制作、プラットフォーム連携。

## 概要

このスキルはオンラインコースの企画・制作・販売をサポートします。
決済処理は各プラットフォーム（Udemy, Teachable等）が行うため、
直接的な金銭処理は含みません。

## コース設計

### ニッチ選定

```
高収益ニッチ:
├── テクノロジー（プログラミング、AI等）
├── ビジネス（マーケティング、起業等）
├── クリエイティブ（デザイン、動画等）
└── 自己啓発（語学、資格等）
```

### カリキュラム構造

```
標準構造:
├── イントロ: コース概要、講師紹介
├── Module 1-N: 各トピック
│   ├── 理論動画
│   ├── 実践動画
│   ├── クイズ/課題
│   └── 補足資料
└── まとめ: 総復習、次のステップ
```

## コンテンツ制作

### フェイスレス動画制作

```
ツール:
├── アバター: HeyGen, Synthesia
├── 録画: OBS Studio
├── 編集: DaVinci Resolve
├── ナレーション: ElevenLabs
└── BGM: 著作権フリー音源
```

### 制作フロー

```
Week 1-2: 企画・設計
Week 3-4: 撮影・収録
Week 5: 編集
Week 6: 仕上げ
Week 7: 公開
```

### ローンチ戦略

```

├── ティーザー動画

├── 早期割引
├── 無料プレビュー
└── SNS集中投稿

Post-launch:
├── レビュー依頼
├── アフィリエイト募集
└── アップセル
```

## 分析・最適化

```
Key Metrics:
├── 登録者数
├── 完了率
├── レビュースコア
└── 返金率

改善アクション:
├── 低登録 → マーケティング強化
├── 低完了率 → コンテンツ改善
└── 低評価 → フィードバック対応
```

---

## 失敗分析・学習

```
追跡項目:
├── 売れないコース → ニッチ再検討
├── 低完了率 → コンテンツ改善
├── 低評価 → 品質向上
└── 返金 → 期待値調整

学習データ保存:
path: /moltworker/data/course-learning.json
```
