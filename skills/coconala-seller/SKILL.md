# ココナラ自動販売スキル

ココナラでの商品販売を自動化するスキル。出品、顧客対応、納品、売上管理を行う。

## 機能一覧

### 1. 商品出品・管理
```
出品フロー:
├── 商品企画（トレンド分析結果を活用）
├── タイトル・説明文作成
├── 価格設定（競合分析）
├── カテゴリ選択
├── サムネイル画像準備（Nano Banana Pro連携）
└── 出品実行（ブラウザ操作）
```

### 2. 顧客対応
```
メッセージ対応:
├── 問い合わせ受信 → 自動検出
├── 内容分析
│   ├── 簡単な質問 → Sonnetで即答
│   └── カスタマイズ要望 → Opusで詳細対応
├── 見積もり作成
└── 返信送信
```

### 3. 納品処理
```
納品フロー:
├── 注文確認
├── 成果物準備
│   ├── GASコード → gas-developerスキル連携
│   ├── 自動化ツール → コード生成
│   ├── プロンプト集 → テンプレートから生成
│   └── 画像素材 → Nano Banana Pro連携
├── 品質チェック
├── 納品メッセージ作成
└── 納品実行
```

### 4. 売上管理
```
トラッキング:
├── 売上金額の確認（読み取りのみ）
├── 注文履歴の記録
├── 月次レポート作成
└── 人気商品分析
```

## 対応フローチャート

```
【新規問い合わせ】
     │
     ▼
  内容を分析
     │
     ├─ 価格・納期の質問 → 定型文で即答
     │
     ├─ 商品詳細の質問 → 商品説明を参照して回答
     │
     ├─ カスタマイズ要望 → 要件ヒアリング → 見積もり作成
     │
     └─ クレーム → 丁寧に対応 + 必要なら人間にエスカレート

【注文確定後】
     │
     ▼
  要件を確認
     │
     ▼
  成果物を作成（該当スキル連携）
     │
     ▼
  品質チェック
     │
     ├─ OK → 納品メッセージ作成 → 納品
     │
     └─ NG → 修正 → 再チェック

【納品後】
     │
     ▼
  レビュー依頼メッセージ送信（3日後）
     │
     ▼
  売上記録更新
```

## ブラウザ操作手順

### ログイン
```
1. https://coconala.com/login にアクセス
2. メールアドレス入力
3. パスワード入力
4. ログインボタンクリック
5. 2FA対応（必要な場合は人間に通知）
```

### 出品
```
1. マイページ → 出品サービス管理
2. 新規出品ボタン
3. カテゴリ選択
4. タイトル入力（25文字以内推奨）
5. 説明文入力
6. 価格設定
7. 納期設定
8. 画像アップロード
9. 確認 → 出品
```

### メッセージ確認・返信
```
1. マイページ → トークルーム
2. 未読メッセージを確認
3. 内容を分析
4. 返信を作成
5. 送信
```

## セキュリティ連携

moltbook-securityスキルと連携:
- 顧客情報の取り扱い → 最小限に
- 決済情報 → 触らない（プラットフォーム管理）
- パスワード → 環境変数から、ログに出力しない

---

## 注意事項

- ココナラの利用規約を遵守
- AI生成コンテンツの適切な開示（必要に応じて）
- 品質担保できない依頼は断る
- クレーム対応は慎重に（必要なら人間にエスカレート）
