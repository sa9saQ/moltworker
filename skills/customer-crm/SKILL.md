# 顧客管理（CRM）スキル

リピーター管理、顧客履歴、関係構築を行うスキル。

## 概要

### なぜ必要か
```
目的:
├── リピーターを識別・優遇
├── 顧客ごとの好みを記憶
├── 過去のやりとりを参照
├── 良い関係を長期的に構築
└── 売上の安定化
```

---

## 顧客データ

### 顧客プロファイル
```
記録する情報:
├── 顧客ID（匿名化ハッシュ）
├── プラットフォーム（ココナラ/Fiverr/Moltbook）
├── 初回取引日
├── 取引回数
├── 累計売上
├── 好みの連絡スタイル
├── 過去の依頼内容
├── 特記事項
└── 満足度（レビュー/フィードバック）
```

### 顧客ランク
```
ランク分け:
├── 新規: 初めての取引
├── リピーター: 2-4回の取引
├── VIP: 5回以上 or 累計¥50,000以上
└── 休眠: 3ヶ月以上取引なし
```

---

## 対応の差別化

### VIP対応
```
VIP顧客への特別対応:
├── 返信優先度を上げる
├── 少し柔軟な納期対応
├── 簡単な追加作業は無料で
├── 新サービスの優先案内
└── 感謝のメッセージ
```

### リピーター対応
```
リピーター顧客への対応:
├── 「いつもありがとうございます」
├── 前回の内容を踏まえた提案
├── 好みに合わせたコミュニケーション
└── リピート割引（状況に応じて）
```

---

## 履歴管理

### 取引履歴
```
記録:
├── 依頼内容
├── 納品物
├── やりとりの要約
├── 良かった点
├── 改善点
└── 次回への申し送り
```

### 好みの記録
```
パーソナライゼーション:
├── 連絡頻度の好み
├── 説明の詳しさの好み
├── 修正への対応スタイル
└── 特別な配慮事項
```

---

## 休眠顧客の掘り起こし

### 再アプローチ
```
3ヶ月以上取引がない顧客:
├── 新サービスの案内
├── 「お元気ですか」的なメッセージ
└── ただし押し付けにならないように
```

---

## プライバシー

### データ保護
```
ルール:
├── 個人情報は保存しない（名前/連絡先等）
├── 匿名化されたIDで管理
├── プラットフォーム外に持ち出さない
├── SNSに顧客情報を投稿しない
└── 顧客の同意なくデータを使用しない
```
