# LinkedIn Automation

LinkedInグロース自動化。投稿、ネットワーク拡大、リード獲得。

## 機能

### コンテンツ投稿
- テキスト投稿
- 画像/カルーセル投稿
- 記事公開
- スケジュール投稿

### ネットワーキング
- 接続リクエスト送信
- メッセージ送信
- コメント/リアクション
- プロフィール閲覧

### リードジェネレーション
- ターゲットリスト作成
- アウトリーチシーケンス
- フォローアップ管理

## 使用方法

### 投稿作成
```
「LinkedInに投稿を作成
 トピック: [topic]
 トーン: [professional/casual/thought-leadership]」
```

### ネットワーキング
```
「[業界]の[役職]に接続リクエスト送信」
「新規接続者にウェルカムメッセージ」
```

### リード獲得
```
「[条件]に合うリードリストを作成」
「[リスト]にアウトリーチ開始」
```

## コンテンツ戦略

### 投稿タイプ

**ストーリー投稿**
```
[フック - 1行目で引き込む]

[ストーリー展開]
- 課題
- 解決
- 学び

[キーテイクアウェイ]

[CTA]
```

**リスト投稿**
```
[タイトル/フック]

1. [ポイント1]
2. [ポイント2]
3. [ポイント3]
...

[まとめ]

👇 コメントで教えてください
```

**Carousel投稿**
```
Slide 1: フック（問題提起）
Slide 2-8: コンテンツ
Slide 9: まとめ
Slide 10: CTA
```

### 投稿頻度
```
理想: 週3-5回
最低: 週1回
時間: 平日朝7-9時, 昼12-13時
```

## ネットワーキング戦略

### 接続リクエストメッセージ
```
[名前]さん

[共通点/きっかけ]について拝見しました。
[理由]から、ぜひつながりたいと思いました。

よろしくお願いいたします。
```

### フォローアップ
```
Day 1: 接続リクエスト
Day 3: 接続後のお礼メッセージ
Day 7: 価値提供（記事/情報共有）
Day 14: カジュアルな質問
Day 30: ミーティング提案（該当時）
```

## リードジェネレーション

### ターゲティング
```
フィルター:
- 業界
- 会社規模
- 役職
- 地域
- キーワード
```

### アウトリーチシーケンス
```
Message 1: 接続 + 自己紹介
Message 2: 価値提供（3日後）
Message 3: ケーススタディ（7日後）
Message 4: CTA（14日後）
```

## アルゴリズム対策

### エンゲージメント向上
```
1. 最初の1時間が重要
2. コメントを促す（質問で終わる）
3. 返信には必ず反応
4. ハッシュタグは3-5個
5. 外部リンクは控えめに
```

### 避けるべきこと
```
- 過度な自動化
- スパム的なDM
- 無関係なハッシュタグ
- 即座の売り込み
```

## 注意事項

### LinkedIn利用規約
```
⚠️ 自動化ツールは規約違反の可能性
⚠️ アカウント制限リスク
⚠️ 日次アクション上限あり:
   - 接続リクエスト: 100/週
   - メッセージ: 150/日
   - プロフィール閲覧: 500/日
```

### 推奨アプローチ
```
- 手動と組み合わせ
- 人間らしいペース
- 価値提供を優先
- パーソナライズ必須
```
